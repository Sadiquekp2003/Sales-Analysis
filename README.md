# Sales-Analysis
The Sales Data Analysis project focuses on transforming raw sales data into meaningful business insights using Power BI. The goal of this project is to help stakeholders understand sales performance, customer behavior, product trends, and revenue patterns through interactive dashboards and visual analytics.

Insights

Top countries: USA, Canada, France.

Lowest among major markets: Mexico.

Most profitable product: Paseo.

Least profitable: Carretera.

Sales trend is seasonal with peaks in mid and late 2014.
